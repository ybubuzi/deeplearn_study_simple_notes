# Python列表

"""
张量(Tensor)概念解释：
- 张量是多维数组的泛化概念

- 1维张量：向量（一维数组）

X_train = np.random.uniform(-1, 1, 100).astype(np.float32).reshape(-1, 1)

## 📖 核心概念速查

### Python基础概念

| Python概念 | Java对比 | JavaScript对比 | 说明 |
|------------|----------|----------------|------|
| `list = [1,2,3]` | `int[] = {1,2,3}` | `array = [1,2,3]` | 列表/数组 |
| `dict = {'a':1}` | `Map<String,Integer>` | `object = {a:1}` | 字典/对象 |
| `for i in range(10)` | `for(int i=0;i<10;i++)` | `for(let i=0;i<10;i++)` | 循环 |
| `if condition:` | `if(condition){` | `if(condition){` | 条件语句 |
| `def func():` | `public void func()` | `function func()` | 函数定义 |

### 深度学习概念

| 概念 | 简单理解 | 代码示例 |
|------|----------|----------|
| **张量(Tensor)** | 多维数组，深度学习的基本数据结构 | `tf.constant([1,2,3])` |
| **层(Layer)** | 神经网络的组成单元 | `tf.keras.layers.Dense(64)` |
| **模型(Model)** | 多个层的组合 | `tf.keras.Sequential([layer1, layer2])` |
| **编译(Compile)** | 设置训练参数 | `model.compile(optimizer='adam')` |
| **训练(Fit)** | 让模型从数据中学习 | `model.fit(X, y, epochs=10)` |
| **预测(Predict)** | 使用训练好的模型 | `model.predict(X_test)` |
| **卷积(Conv2D)** | 提取图像局部特征 | `tf.keras.layers.Conv2D(32, (3,3))` |
| **池化(MaxPool)** | 降低图像分辨率 | `tf.keras.layers.MaxPooling2D((2,2))` |
| **激活函数** | 增加非线性 | `activation='relu'` |
| **Dropout** | 防止过拟合 | `tf.keras.layers.Dropout(0.5)` |

### 时序模型概念

| 概念 | 简单理解 | 应用场景 |
|------|----------|----------|
| **时序数据** | 按时间顺序的数据 | 股价、天气、销量 |
| **滑动窗口** | 用过去N个时间点预测下一个 | 用过去7天预测明天 |
| **RNN** | 有记忆的神经网络 | 简单序列处理 |
| **LSTM** | 更强记忆能力的RNN | 长序列、文本处理 |
| **词嵌入** | 将单词转换为向量 | 文本分析的基础 |

## 🛠️ 常见问题解答

### Q1: Python语法看不懂怎么办？
**A**: 先看`Python语法速查表_for_JS_Java开发者.md`，里面有详细的语法对比。

### Q2: 为什么要用NumPy？
**A**: NumPy提供高效的数组运算，类似于Java的数学工具类，但功能更强大。

### Q3: TensorFlow和Keras是什么关系？
**A**: Keras是TensorFlow的高级API，类似于jQuery之于JavaScript，让深度学习更简单。

### Q4: 什么是张量？
**A**: 张量就是多维数组，0维=数字，1维=数组，2维=矩阵，3维=立体数组。

### Q5: 为什么需要数据标准化？
**A**: 不同特征的数值范围差异很大，标准化确保模型能平等对待所有特征。

## 💡 学习技巧

### 1. 对比学习法
每当遇到Python新语法，立即想想Java/JS中的对应写法：
```python
# Python
for item in list:
    print(item)

# 对比思考：
# Java: for(Item item : list) { System.out.println(item); }
# JS: for(let item of list) { console.log(item); }
```

### 2. 注释驱动学习
看代码时先看注释，理解概念后再看代码：
```python
# 1. 先看这个注释理解概念
"""
LSTM：长短期记忆网络，比RNN更强的记忆能力
"""

# 2. 再看代码理解实现
tf.keras.layers.LSTM(64, dropout=0.5)
```

### 3. 实验验证法
修改参数看结果变化：
```python
# 原始参数
model.fit(X, y, epochs=10, batch_size=32)

# 尝试修改
model.fit(X, y, epochs=50, batch_size=16)  # 看训练效果变化
```
